READ ME

Para usar este compilador de ply ir al directorio en donde descargo los archivos:
	•	MyP.py
	•	MyPLex.py
	•	MyRParse.py

En la consola o shell del directorio de estos archivos escribir: python3 MyR.py text1.txt
esta instrucción abrirá la línea de comandos de MyR.

MODO INTERACTIVO
[MyR] >
Aquí es donde puede escribir las instrucciones y obtener resultados:
Ejemplos:
[MyR] > 2 + 2
4
[MyR] > A = 1
[MyR] > B = 4
[MyR] > C = 1
[MyR] > aux1 = -B / (2 * A)
[MyR] > aux2 = B ^ 2 – (4 * A * C)
[MyR] > if aux2 > 0 then aux2 else -aux2
12
